GraphQL

A query language that allows clients to request and retrieve specific data from a server.

Compared to REST

RESTful APIs have fixed endpoints that result in a simpler data structures; meanwhile GraphQL:

  • produces customizable data structures
  • prevents over & underfetching of data
  • requires less load from devices
