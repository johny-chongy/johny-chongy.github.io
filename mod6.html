An easy trivia game intended to pass the time; users are able to customize settings for topics and number of players. Used TypeScript so that the Game Settings followed a specific interface in order to prevent inappropriate settings. Also leveraged localStorage to cache simple settings 90% faster than utilizing a database.

Utilizing useEffect hook to retrieve current Game State 

                    
                    useEffect(function retrieveGameState() {
                        console.log('useEffect');
                        async function startGame() {
                            const localGameState = localStorage.getItem('gameState') || '';
                            if (localGameState !== '') {
                                const parsedGameState = JSON.parse(localGameState);
                                const remainingStart = parsedGameState.remaining === 0 ? -1 : parsedGameState.remaining;
                                setGameState(state => ({
                                    ...parsedGameState,
                                    remaining: remainingStart
                                }));
                                setIsLoading(false);
                            }

                            setIsLoading(false);
                        }
                        startGame();
                    }, []);
